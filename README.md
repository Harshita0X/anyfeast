# AnyFeast

AI-powered personalized meal planning and nutrition tracking for your health goals.

## Features

- **Personalized Onboarding** - Multi-step onboarding flow to understand your goals, dietary preferences, and health metrics
- **Smart Meal Planning** - AI-generated meal plans based on your preferences, allergies, and budget
- **Nutrition Tracking** - Track your daily nutrition intake and progress towards your goals
- **Diet Customization** - Support for various diets including Vegetarian, Vegan, Keto, Jain, and more

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm

### Installation

```bash
# Clone the repository
git clone https://github.com/Harshita0X/anyfeast.git

# Navigate to project directory
cd anyfeast

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:8080`

## Tech Stack

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Animations**: GSAP
- **Build Tool**: Vite
- **State Management**: React Query

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run test` - Run tests
